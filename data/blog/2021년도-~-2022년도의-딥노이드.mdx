# **주요 업무 1: [관세청 불법 복제품 판독 시스템](https://m.blog.naver.com/PostView.naver?blogId=deepnoid&logNo=222784301583&referrerCode=0&searchKeyword=%EA%B4%80%EC%84%B8%EC%B2%AD)**

출처 : 이데일리 뉴스 참조

## 사용자 인터렉션 속도 : 불법 복제품 판독 기능 개선

- **문제**: 기존 시스템에서 **복제품 판독 속도가 느리고**, 판별 과정의 신속성과 정확성이 부족하여 사용자들이 불편함을 느꼈습니다.
- **해결**: **카메라 확대/축소 및 이미지 크롭 기능**을 추가하여 사용자들이 복제품의 미세한 특징을 직접 검토할 수 있도록 개선했습니다. **React와 비동기 처리 기술**을 활용해 **최소 5개의 이미지**를 비교 분석할 수 있는 인터페이스를 구현했습니다.

## 데이터 연계 및 관리 : 아코디언 UI를 통한 데이터 표시

- **문제**: 다량의 판독 데이터를 **효율적으로 관리**하고, 사용자가 직관적으로 탐색할 수 있는 UI가 필요했습니다.
- **해결**: **React와 부트스트랩을 사용한 아코디언 UI**로 상위 5개 판독 결과를 요약 표시하고, **모바일 웹뷰에서 직관적 공간 활용 최적화**로 모바일 사용자 경험을 개선했습니다.
- **성과**: 사용자 탐색 시간 **40% 단축** 및 작업 효율성 **대폭 향상**.

# **주요 업무 2 : [KAC DEEP SECURITY](https://m.blog.naver.com/deepnoid/222753739088)  (장비 납품 진행)**

출처 : 세콘 사진 촬영 및 아시아투데이 뉴스 일부 캡쳐

## 리팩토링 및 성능 최적화

- **문제**: **바닐라 자바스크립트 기반 성능**이 부족하여 사용자들이 **느린 로딩 속도와 반응성에 대한 불만**을 제기했습니다.
- **해결**: **React 기반의 SPA 전환**으로 클라이언트 측에서 필요한 리소스를 한 번만 로드하여 **즉각적인 반응** 구현 및 **비동기 데이터 로딩 최적화**로 데이터 로딩 시간을 최소화했습니다.
- **성과**:

## 실시간 데이터 처리 : 비디오 판독 데이터의 실시간 반영

- **문제**: **비디오 판독 데이터를 실시간**으로 사용자에게 제공해야 했으나, 기존 시스템은 즉각적인 업데이트와 비교 기능이 제한적이었습니다.
- **해결**: **WebSocket을 활용한 양방향 통신**으로 **실시간 데이터 반영** 및 **데이터 비교 기능**을 구현하여 사용자들이 세부 정보를 더 빠르게 분석할 수 있도록 개선했습니다.
- **성과**: **사용자 편의성**과 **데이터 신뢰성** 크게 향상.

## 데이터 시각화 : 직관적인 데이터 분석 환경 구축

- **문제**: 사용자들이 데이터를 효율적으로 이해하고 **빠르게 분석**할 수 있는 시각화 도구가 필요했습니다.
- **해결**: **통계 라이브러리를 활용한 다양한 그래프**(직선 그래프, 도넛 그래프, 원형 그래프 등) 추가 및 **사용자 정의 가능한 통계 옵션**으로 탐지 비율이 높은 케이스를 쉽게 식별하도록 했습니다.
- **성과**: 업무 효율성 향상 및 **데이터 해석 용이성 증대**.

## 고객 요구 사항 반영 및 맞춤형 설정 기능 추가

- **문제**: **다양한 고객 요구를 충족할 커스터마이즈 기능**과 지속적인 유지보수가 필요했습니다.
- **해결**: **카메라 위치, 통계 설정, 2D/3D 비디오 판독** 등의 옵션을 추가해 **맞춤형 설정 기능 강화**. **주기적인 유지보수**를 통해 시스템 안정성 확보.
- **성과**: 사용자 만족도와 시스템 유연성 및 안정성 **동시 확보**.

반응형 웹 디자인의 기본 원칙을 이해하고, 미디어 쿼리를 사용해 다양한 화면 크기에 대응하는 방법을 실무에서 어떻게 적용했는지를 평가하려는 것입니다. 이를 통해 유연한 레이아웃 설계 능력과 디자인 및 사용자 경험(UX)에 대한 이해도를 확인할 수 있습니다.

---

### 모범 답안

**반응형 웹 디자인(Responsive Web Design)의 기본 원칙:**

1. **유연한 그리드 레이아웃(Flexible Grid Layout)**: 반응형 웹 디자인의 핵심은 레이아웃이 고정되지 않고, 화면 크기에 따라 유동적으로 변하는 그리드 시스템을 사용하는 것입니다. 퍼센트(%)나 `fr` 단위를 사용하여 각 요소의 크기를 화면 크기에 맞게 조정할 수 있습니다.
2. **유연한 이미지 및 미디어(Flexible Images and Media)**: 이미지와 비디오와 같은 미디어 파일도 화면 크기에 맞춰 크기가 조절되어야 합니다. `max-width: 100%` 같은 규칙을 사용하여, 이미지가 화면을 넘어서지 않도록 조정합니다.
3. **미디어 쿼리(Media Queries)**: 미디어 쿼리는 CSS에서 화면의 크기에 따라 스타일을 다르게 적용할 수 있도록 하는 기술입니다. 이를 통해, 같은 콘텐츠라도 데스크톱, 태블릿, 모바일 환경에서 각각 다른 레이아웃을 제공할 수 있습니다.

**미디어 쿼리 적용 방법:**

```css
/* 기본 스타일: 작은 화면용 */
body {
  font-size: 14px;
  margin: 0;
}

.container {
  width: 100%;
  padding: 10px;
}

/* 중간 크기 이상의 화면용 스타일 */
@media (min-width: 768px) {
  body {
    font-size: 16px;
  }

  .container {
    width: 750px;
    margin: 0 auto;
  }
}

/* 큰 화면용 스타일 */
@media (min-width: 1200px) {
  body {
    font-size: 18px;
  }

  .container {
    width: 1170px;
    margin: 0 auto;
  }
}
```

- **min-width**를 사용하여 특정 크기 이상의 화면에서 레이아웃을 변경합니다. 예를 들어, 768px 이상에서는 `.container`의 너비를 고정시키고, 중앙에 정렬합니다.
- 작은 화면에서는 요소들이 화면의 가로를 채우며 유연하게 동작하고, 큰 화면에서는 더 많은 공간을 활용하여 가독성을 높입니다.

**실무에서의 적용 예시:**

- **반응형 네비게이션 메뉴**: 모바일에서는 햄버거 메뉴로 축소하여 사용하고, 데스크톱에서는 전체 메뉴를 보여줍니다.
- **그리드 레이아웃**: 화면 크기에 따라 1~4개의 컬럼으로 유연하게 배치되며, 제품 리스트나 이미지 갤러리와 같은 경우에 자주 사용됩니다.
- **타이포그래피**: 화면이 커질수록 글자 크기를 비율에 맞춰 조절하여 가독성을 유지합니다.

###

웹사이트가 다양한 디바이스에서 동일하게 보이는 것은 과거의 이야기입니다. 오늘날의 웹사이트는 **다양한 화면 크기**에 맞춰 **유연하게 변화**해야 합니다. 이러한 웹사이트를 만드는 기법이 바로 **반응형 웹 디자인(Responsive Web Design)**입니다.

### 1. 반응형 웹 디자인의 기본 원칙

**유연한 그리드 시스템**과 **미디어 쿼리**를 통해 화면 크기와 관계없이 콘텐츠가 적절히 표시되도록 합니다.

1. **유연한 레이아웃**: 픽셀 기반이 아닌 퍼센트(%)나 유닛(fraction)을 사용하여 화면 크기에 맞게 요소들이 재배치됩니다.
2. **유연한 미디어**: 이미지와 비디오가 큰 화면에서는 커지고, 작은 화면에서는 축소됩니다.
3. **미디어 쿼리**: CSS의 강력한 기능으로, 화면 크기나 해상도에 맞는 스타일을 적용할 수 있습니다.

### 2. 미디어 쿼리 활용하기

미디어 쿼리는 다양한 화면 크기에서 어떻게 디자인이 변화해야 하는지 지정할 수 있는 방법입니다. **기준이 되는 화면 너비**를 설정하여, 그 너비에 맞는 스타일을 적용하는 방식입니다.

```css
/* 작은 화면 기본 스타일 */
.container {
  padding: 10px;
}

/* 태블릿 이상의 화면에 적용 */
@media (min-width: 768px) {
  .container {
    width: 750px;
    margin: 0 auto;
  }
}

/* 데스크톱 화면에 적용 */
@media (min-width: 1200px) {
  .container {
    width: 1170px;
  }
}
```

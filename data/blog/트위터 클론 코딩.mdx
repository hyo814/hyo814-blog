# 트위터 클론 코딩

[https://github.com/hyo814/hwitter](https://github.com/hyo814/hwitter)

### **프로젝트 개요**

- **프로젝트 기간**: 2020년 12월 25일 ~ 12월 29일
- **사용 기술**: React, Firebase
- **프로젝트 목적**: 트위터와 유사한 소셜 네트워크 기능을 가진 웹 애플리케이션을 개발하여, 프론트엔드 기술과 Firebase를 활용한 백엔드 기술을 학습

### **구현 기능**

1. **회원 기능**:
    - **로그인, 회원가입, 로그아웃** 기능을 구현하여 사용자가 개별 계정을 통해 접근 가능
    - Firebase를 통해 소셜 로그인 및 인증 기능을 구축
2. **실시간 소통**:
    - Firebase를 활용해 실시간으로 사용자와 상호작용할 수 있는 기능을 추가
    - 트위터와 유사하게 텍스트 게시글을 작성하고 소통 가능
3. **프로필 관리**:
    - 사용자 프로필 관리 기능을 구현 (추후 보완 예정)
    - 사용자가 프로필 정보를 수정할 수 있도록 설계

### **프로젝트 성과 및 학습 내용**

- **React의 다양한 활용**:
    - React를 활용해 다양한 컴포넌트 구조와 상태 관리를 연습
    - 사용자 인터페이스 구성과 재사용 가능한 컴포넌트 설계에 대한 이해도 향상
- **Firebase의 구체적 활용**:
    - Firebase를 이용한 데이터베이스 구성과 실시간 데이터 통신 구현
    - 소셜 로그인과 같은 인증 기능과 데이터의 실시간 연동 학습
- **호스팅 경험**:
    - Firebase 호스팅 시도 중 계정 관련 문제(예전 계정 미결제 오류)로 완벽히 적용하지 못했으나, 실제 배포 과정에서 발생하는 문제를 파악하고 대응 경험을 쌓음

### **프로젝트 회고 및 개선점**

- **프로필 기능 보완 필요**: 프로필 부분에서 일부 기능이 누락되어 추후 더 깊이 공부하면서 보완할 계획
- **기술 확장성 고려**: 트위터 기능 구현을 통해 실시간 소통과 인증 시스템에 대한 이해도를 높였으며, 향후 다른 SNS 기능을 추가하거나 확장하는 방향으로 발전 가능성을 염두에 둠

---

이 프로젝트는 단순한 클론 코딩을 넘어 실제 소셜 네트워크 서비스의 주요 기능을 체계적으로 구현하고, Firebase와의 통합을 통해 프론트엔드와 백엔드 기술을 종합적으로 학습하는 계기가 되었습니다. 향후 추가 기능과 개선점을 통해 더욱 완성도 높은 애플리케이션으로 발전시킬 계획입니다.

### **프로젝트 개요**

- **프로젝트 기간**: 4월 ~ 7월 (공동 학습 및 프로젝트 진행), 7월 이후 직접 개발 (~ 12월)
- **사용 기술**: React, GitHub, Code Review
- **개발 기능**: 회원가입, 로그인, 마이페이지, 마이페이지 수정, 계정 탈퇴 기능

### **프로젝트 경험 및 성과**

프론트엔드 개발에 대한 배경지식이 거의 없는 상태에서 시작했지만, 팀원들과의 협력을 통해 단계적으로 학습해 나가며 프로젝트를 성공적으로 이끌었습니다. 특히 GitHub를 사용한 코드 리뷰와 병목 현상 해결 과정을 통해 협업의 중요성을 실감하게 되었고, 실제 개발 경험을 통해 React에서의 기초적인 사용자 인증 기능을 완성할 수 있었습니다.

1월부터는 대면 근무로 전환되어 프로젝트를 마무리하지는 못했지만, 제가 맡았던 영역은 충분히 구현하여 팀에 기여했습니다.

### **개발 과정에서의 도전과 성장**

- **팀원과의 공동 학습**: 4월부터 7월까지 프론트엔드 기초를 팀원들과 함께 공부하며, 서로의 성장을 도왔습니다. 초기에는 게시판 기능도 구현할 예정이었지만, 프레임워크에 대한 이해도를 고려하여 사용자 인증 관련 기능에 집중했습니다.
- **협업을 통한 코드 품질 개선**: GitHub에서의 코드 리뷰를 통해 코드의 품질을 높였고, 병목 현상과 같은 문제를 협업으로 해결하는 경험을 통해 기술적 성장을 이룰 수 있었습니다.
- **다양한 역할 병행**: 21학점 수강과 바이오 데이터 재택근무와의 병행 속에서도 프로젝트를 지속해온 경험은 문제 해결 능력을 더욱 강화하는 계기가 되었습니다.

### **다음 목표**

이번 프로젝트를 통해 얻은 경험을 바탕으로 더 다양한 프레임워크와 기술 스택을 공부할 계획입니다. 앞으로는 React뿐만 아니라, 응용 가능한 다양한 프레임워크 학습을 통해 프론트엔드 개발자로서의 역량을 확장하고자 합니다.

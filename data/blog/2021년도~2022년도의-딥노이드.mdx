## **주요 업무 1: [관세청 불법 복제품 판독 시스템](https://m.blog.naver.com/PostView.naver?blogId=deepnoid&logNo=222784301583&referrerCode=0&searchKeyword=%EA%B4%80%EC%84%B8%EC%B2%AD)**

![딥노이드](/static/images/deepnoid/image3.webp)
출처 : 이데일리 뉴스 참조

기술 스택 : JavaScript, React , Redux, Bootstrap

### 1. 사용자 인터렉션 속도 : 불법 복제품 판독 기능 개선

- **문제**:

  - 기존 시스템에서 **복제품 판독 속도가 느리고**, 판별 과정의 신속성과 정확성이 부족하여 사용자들이 불편함을 느낌.

- **문제 해결**:

  - **카메라 확대/축소 및 이미지 크롭 기능** 추가로 사용자가 복제품의 미세한 특징을 직접 확인 가능.
  - **React**와 **비동기 처리 기술**을 활용하여 **최소 5개의 이미지**를 비교 분석할 수 있는 인터페이스 구현.

- **성과**:
  - 카메라 촬영 시 확대/축소와 이미지 크롭 기술 도입으로 불법 복제품 분석의 정확성 및 신뢰성 향상.
  - 사용자 경험(UX) 개선, 판독 결과의 신뢰도 증가.

### 2. 데이터 연계 및 관리 : 아코디언 UI를 통한 데이터 표시

- **문제**:

  - 다량의 판독 데이터를 효율적으로 관리하고, 사용자가 직관적으로 탐색할 수 있는 UI 필요.

- **해결**:

  - **React**와 **부트스트랩** 기반의 **아코디언 UI**로 상위 5개 판독 결과를 요약하여 표시.
  - **모바일 웹뷰 최적화**로 공간 활용을 극대화하며, 모바일 사용자 경험 개선.

- **성과**:
  - 사용자 탐색 시간 **40% 단축**, 작업 효율성 **대폭 향상**.
  - 중요한 정보를 간결하게 표시하고, 필요 시 세부 정보를 확장할 수 있는 구조로 사용자 경험과 모바일 사용성 개선.

## **주요 업무 2 : [KAC DEEP SECURITY](https://m.blog.naver.com/deepnoid/222753739088) (장비 납품 진행)**

![딥노이드](/static/images/deepnoid/image.gif)
![딥노이드](/static/images/deepnoid/image2.webp)
출처 : 세콘 사진 촬영 및 아시아투데이 뉴스 일부 캡쳐

기술 스택 : JavaScript, React, Redux, Bootstrap

### 1. **사용자 정의 설정 기능 추가**

- **문제**:
  - 다양한 사용자 요구를 충족할 커스터마이즈 기능 필요.
- **해결**:
  - 카메라 위치, 통계 설정, 2D/3D 비디오 판독 등 다양한 옵션 추가.
- **성과**:
  - 사용자가 설정을 유연하게 조정 가능. 사용자 만족도 증가 및 회사별 커스텀 요청 대폭 증가.

### 2. **웹 소켓 통신과 비디오 판독 기능 구현**

- **문제**:
  - 비디오 판독 데이터를 실시간으로 사용자에게 제공할 필요.
- **해결**:
  - **웹 소켓**을 활용한 양방향 통신으로 실시간 데이터 전송.
  - 단화면 및 양화면 모드 구현, 줌/확대 기능 추가로 편의성 향상.
- **성과**:
  - 데이터 실시간 반영 및 탐지 정확도 증가, 사용자 분석 효율성 향상.

### 3. **SPA 구조 전환으로 성능 및 사용자 경험 개선**

- **문제**: 기존 바닐라 JavaScript 기반 성능 부족으로 느린 로딩 속도와 제한된 사용자 경험 발생.
- **해결**:
  - **React 기반 SPA**로 전환, 클라이언트 측 렌더링으로 성능 개선.
  - 컴포넌트 기반 아키텍처 도입으로 개발 효율성 및 유지보수 용이성 증가.
- **성과**:
  - 빠른 페이지 탐색, 즉각적인 피드백 제공, 사용자 만족도 및 생산성 향상.

### 4. **데이터 시각화: 직관적인 분석 환경 구축**

- **문제**:
  - 비디오 판독 데이터를 기반으로 금지 품목의 패턴 분석 필요.
- **해결**:
  - 다양한 그래프(직선, 도넛, 원형) 추가로 데이터 시각화 다양화.
  - 사용자 정의 가능한 통계 조건 설정 기능 구현.
- **성과**:
  - 데이터 분석 효율성 증대 및 탐지 정확도 향상.

### 5. **세계 보안 엑스포(SECON) 2021~2022 참여**

- **성과**:
  - 최신 보안 기술 발표 및 기술 교류를 통한 네트워킹 확대.
  - 새로운 프로젝트 문의 및 기획 추진 기회 창출.

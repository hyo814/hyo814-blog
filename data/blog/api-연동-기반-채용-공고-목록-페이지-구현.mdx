### 🔍 과제 개요
- **목표:** API 연동 기반 채용 공고 목록 페이지 구현
- **기간:** 2024년 1월 21일 ~ 1월 28일
- **기술 스택:** Next.js, TypeScript, Tailwind CSS
- **핵심 요구 사항:**
  - API에서 데이터를 불러와 최신순으로 정렬
  - 등록일 기준으로 ‘Today’, ‘n days ago’ 또는 ‘YYYY.MM.DD’로 날짜 표시
  - 실시간 검색 기능 구현 (제목 및 키워드 기준)
  - 오류 처리 및 예외 상황 대응

### 📌 프로젝트 구조
```
├── client            # 프론트엔드 작업 디렉토리
│   ├── components    # 컴포넌트 폴더
│   ├── pages         # Next.js 페이지 폴더
│   ├── styles        # Tailwind 기반 스타일 설정
│   ├── utils         # 유틸리티 함수 및 API 호출 관리
│   ├── README.md     # 문서화 파일
├── markup            # 기존 제공된 HTML/CSS 마크업
├── result            # 결과 이미지 폴더
├── server            # API 제공을 위한 Node.js 서버
└── README.md         # 프로젝트 전체 문서화
```

### ⚙️ 개발 과정
#### 1. **Next.js 프로젝트 세팅**
- `npx create-next-app@latest client`
- `tailwindcss` 및 `eslint` 설정
- API 연동을 위한 `axios` 설치 및 설정

#### 2. **API 연동 및 데이터 정렬**
- `fetchJobs` 함수에서 `GET /rest/v1/jobs` 호출 후 최신순 정렬 적용
- 날짜 포맷 처리: `dayjs` 활용하여 상대 날짜 표기 (Today, n days ago)

#### 3. **검색 기능 구현**
- `useState` 및 `useEffect`를 활용한 실시간 검색
- `input` 변경 시 즉시 필터링되도록 설정

#### 4. **UI 구성**
- 기존 `index.html`, `style.css` 기반으로 Tailwind 적용
- 헤더, 리스트, 검색 바 등 컴포넌트화 진행

### 🛠️ 문제 해결 및 예외 처리
- **API 실패 대응:** `try-catch` 및 `error state` 추가
- **검색어 대소문자 구분 문제:** `.toLowerCase()` 처리 후 검색
- **Next.js SSR 관련 이슈:** `useEffect` 활용하여 CSR 방식 적용

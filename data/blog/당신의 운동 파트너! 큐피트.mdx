# 당신의 운동 파트너! 큐피트

**[https://github.com/schoolproject2020](https://github.com/schoolproject2020)**
****

### 프로젝트 개요

본 프로젝트는 **React**, **HTML**, **CSS**를 활용하여 프론트엔드를 구축하고, **Flask**와 웹 크롤링을 통해 백엔드 데이터를 수집하여 완성된 웹 애플리케이션입니다.

### 주요 역할

저는 프론트엔드를 담당하여 사용자 경험(UI/UX)을 최적화하는 데 집중하였으며, UI/UX 디자이너와 협업하여 일관된 사용자 흐름을 구현했습니다. 특히, **Redux**를 활용하여 상태 관리를 효율적으로 처리하였으며, 코드 품질을 높이기 위해 필요한 부분에서 오류를 탐색하고 해결하는 데 힘썼습니다.

### 기능 및 구성 요소

- **메인 페이지**: 사용자 친화적인 메인 화면 구성
- **로그인 시스템**: 사용자 인증 및 권한 관리
- **고객 관리 시스템**: 고객 데이터를 효율적으로 관리하고 추적
- **운동, 패션, 식단 관리 기능**: 각 기능별로 사용자 맞춤화된 데이터 제공

### 학습과 성장

이 프로젝트를 통해 React와 Redux에 대한 실전 경험을 쌓으며 실제 사용자 인터페이스와 더 밀접하게 연결된 지식을 습득할 수 있었습니다. 빠르게 성장할 수 있었던 프로젝트로, 많은 오류를 해결하며 개발자로서 역량을 넓힐 수 있었습니다.

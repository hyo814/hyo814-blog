### 자기소개서

안녕하세요, 프론트엔드 개발자 임효진입니다.

컴퓨터정보공학부를 졸업한 후, 3년간 프론트엔드 개발자로 활동하며 사용자 경험 중심의 웹 애플리케이션을 설계하고 개발해 왔습니다. 이 과정을 통해 프론트엔드 개발이 단순한 기술 구현을 넘어 사용자와 개발자 간의 소통을 원활하게 하고, 사용자 경험을 개선하며 개발 효율성을 높일 수 있는 매력적인 분야라는 것을 깨달았습니다.

**경력 소개**

저는 스테이폴리오에서 글로벌 사용자를 위한 다국어 지원과 프리오더 및 프로모션 페이지 개발을 담당했습니다. i18n 라이브러리와 OPEN API 기반 번역 기능을 활용하여 다국어 지원을 구현함으로써 글로벌 시장에서의 접근성을 향상시켰습니다. 또한 사용자 행동 데이터를 분석하여 마케팅 전략을 최적화하고, 주기적인 프로모션 페이지 업데이트를 통해 사용자 경험을 지속적으로 개선했습니다.

딥노이드에서는 관세청 불법 복제품 판독 시스템 개발에 참여하여 사용자 친화적인 인터페이스를 구현했습니다. KAC DEEP SECURITY 프로젝트에서는 웹소켓을 활용한 실시간 비디오 판독 기능을 통해 AI 기반 불법 물품 탐지를 구현하고, 사용자 맞춤형 설정 기능을 추가하여 다양한 고객 요구를 충족시켰습니다. 이를 통해 전체 시스템의 효율성을 높이는 데 기여했습니다.

또한, 직짱건강 플랫폼에서는 JWT를 활용한 인증 시스템을 구축하고, React Query를 사용하여 데이터 일관성을 유지하며 비동기 통신을 자동화했습니다. 이 과정에서 최신 기술 트렌드를 학습하고 실전에 적용하여 사용자 데이터의 보안과 신뢰성을 강화하였습니다.

**역량 및 목표**

프론트엔드 개발자로서 저는 사용자 중심의 디자인 사고를 바탕으로 UX를 최우선으로 고려하며 직관적이고 사용하기 쉬운 인터페이스를 설계하고 구현하는 데 중점을 두고 있습니다. 사용자 피드백을 적극 반영하여 UI/UX를 지속적으로 개선하고, 이를 통해 사용자의 만족도를 높이는 데 기여하고 있습니다. 특히, 여행 관광업 관련 프로젝트에서 사용자 경험을 향상시키기 위해 사용자의 여정을 분석하고 UI를 개선하여 방문자 전환율을 높인 경험은 저의 역량을 잘 보여주는 사례입니다.

효율적인 커뮤니케이션과 조율 능력을 통해 다양한 이해관계자들과의 원활한 의사소통을 통해 프로젝트 목표를 명확히 하고, 효과적인 협업을 통해 프로젝트의 성공적인 완수를 이끌어냈습니다. 프로젝트 관리 툴을 활용하여 팀 내 소통을 강화하고, 이슈를 빠르게 해결함으로써 프로젝트 진행 속도를 높였습니다.

저는 변화에 유연하게 대처하고 새로운 기술을 습득하며 발전할 준비가 되어 있습니다. 끊임없는 자기 계발을 통해 최신 기술을 적극적으로 도입하고 활용하여 프로젝트의 품질을 높였습니다. 예를 들어, 최신 프론트엔드 프레임워크와 라이브러리를 습득하여 프로젝트에 적용함으로써 개발 속도와 효율성을 향상시켰습니다.
